- 用户原话：`给这个游戏取个名字吧`

- 命名理由：四个中文字符便于记忆；“防线”直接对应塔防；“零号”保留军事编号感，也能承载“退无可退的最后防区”世界观。
- 撞名初筛：公开搜索暂未发现以“零号防线”为准确标题的主流游戏；这不是商标注册结论。
- 备选方案：`《灰烬哨站》 / ASHEN OUTPOST`、`《终夜防线》 / LAST LIGHT LINE`。
- 用户确认原话：`OK, 那就叫零号防线`
- 状态：⭐ 正式定名为 `《零号防线》 / LINE ZERO`。
- 统一范围：游戏菜单、浏览器标题、README/分享文案、单文件名称、说明文档和 GitHub Pages。
- 兼容边界：保留 `last-outpost-audio-settings-v1`、`last-outpost-v2` 与既有 GitHub URL，避免存档设置丢失、素材路径失效和旧分享链接断开。
