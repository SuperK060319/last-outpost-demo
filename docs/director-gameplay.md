# LINE ZERO：炮塔定位与首关经济审查

## 设计判断

本轮只校准三座炮塔和三条升级线，不改变敌人、路线、波次数量、闸门或资源产出。

- **机枪**负责稳定处理普通目标。旧弹匣升级只把长期输出提高约 2%，玩家花同样价格却几乎感受不到变化；现在换弹间隔更明确，威力、供弹和扩容首级都能改变第一波结果。
- **迫击炮**只在敌人聚集时超过机枪。旧“每轮炮弹”首级直接翻倍，而威力与装填提升明显更小，属于必选项；现在三个方向的同级总输出倍率接近，差异落在单次爆发、开火频率和一轮炮弹数。
- **狙击塔**用于高血量目标和直线穿透。穿透升级只在至少两个目标时兑现收益；面对单个 Boss，威力与锁定速度才是有效选择。

## 首关经济结论

- 初始补给为 42。
- 机枪或迫击炮建造 24，首级强化 18，合计正好 42；玩家开局始终能完成一条明确构筑。
- 狙击塔建造 32，剩余 10；一级补给站持续产出 0.20/秒，并在每波开始运输17点。狙击开局需要先打第一波，再完成首级强化。
- 普通据点若不打开冻结弹窗，18秒部署期约增加3.6补给，第一波前仍不能直接部署两座廉价塔；持续补给主要用于缩短“差几点才能购买”的等待。

## 验收口径

`scripts/balance-sim.mjs` 使用当前第一波的 5 个敌人、58 生命、约 3.45 秒抵达时间和闸门攻击频率，比较“一座机枪 + 任一首级强化”的结果。它不是完整战斗AI，只用于防止三个明显问题回归：

1. 建造后买不起任何首级强化。
2. 同价升级中存在压倒性的必选项。
3. 为平衡炮塔而意外改变第一波敌人或经济节奏。

## 下一批敌人的三个低成本方案（未实现）

### 方案 A：跑尸

- 生命低、速度约普通尸的 1.45 倍。
- 让机枪供弹速度和狙击锁定速度产生价值。
- 只需复用普通尸素材做体型、色调与移动速度变化。

### 方案 B：防爆尸

- 生命高、移动慢，承受迫击炮伤害降低，但不额外减免狙击。
- 让狙击威力成为对高价值目标的明确答案。
- 只需复用 Boss 或普通尸素材叠加盾牌轮廓，不需要新路径。

### 方案 C：尸群小队

- 一次生成 3 个低生命敌人，横向轻微错位，保持同一条直路。
- 直接验证迫击炮范围与多弹升级的成群收益。
- 不增加新AI，仅改变一次刷新数量和单体生命。
