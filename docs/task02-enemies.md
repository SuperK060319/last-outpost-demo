# 任务2：敌人种类扩充

## 内容结构

敌人配置采用三层目录，避免行为尚未实现时污染现有第一关：

- `BASE_ENEMIES`：普通感染体和Boss的完整图鉴配置，场景仍保留现有特殊处理。
- `ENEMIES`：已经接入战斗的跑尸、装甲尸和尸群感染体。
- `EXTRA_ENEMIES`：护盾兵、爆破尸、毒囊尸和强化军医尸。
- `LIVE_ENEMY_CATALOG`：当前五类安全目录。
- `ENEMY_CATALOG`：九类完整目录，供图鉴和未来关卡使用。

纯数据API：

- `getEnemyConfig(type, includeExtra)`：读取敌人定义。
- `enemyTravelSeconds(type, wave)`：获得该类型目标到门时间。
- `enemyStatsForWave(type, wave)`：生成图鉴、预告或模拟用的HP、门伤、奖励、旅行时间与组数量。

## 新增四类敌人

### 护盾兵 `frontal-shield`

- 生命1.25倍、速度0.72倍、门伤0.9倍、奖励1.45倍。
- 正面防暴盾减少55%直射伤害，护盾生命为本体最大生命的35%。
- 巨大矩形青灰盾牌占据正面轮廓，优先用迫击炮、榴弹或火焰绕过直射防护。

### 爆破尸 `gate-detonation`

- 生命0.78倍、速度1.05倍、单次门伤2.4倍、奖励1.3倍。
- 抵达闸门后读条650毫秒并自毁。
- 胸前红色爆破筒和中心警示灯形成识别点；应由机枪或高优先级单体火力提前击杀。

### 毒囊尸 `death-cloud`

- 生命0.9倍、速度0.78倍、门伤0.65倍、奖励1.3倍。
- 死亡留下64px、持续3秒的毒雾，使覆盖炮塔攻击间隔增加25%。
- 背部巨大黄绿色毒囊高过头部；狙击应尽量在远离基地的位置击杀。

### 强化军医尸 `support-pulse`

- 生命0.82倍、速度0.86倍、门伤0.5倍、奖励1.55倍。
- 每1.8秒治疗92px内友军6%最大生命，并提高8%移动速度。
- 方形医疗电台、高天线、旧白识别带和药剂罐组成后排支援轮廓；应设置为狙击高价值目标。

## 程序回退纹理

`src/game/content/extraEnemyTextures.js` 导出 `createExtraEnemyTextures(scene)`，为四类扩展敌人生成独立纹理key。本轮没有连接BootScene和`createTextures.js`，因此不改变当前素材加载。

## 主分支集成顺序

1. BootScene接入扩展回退纹理，正式PNG继续使用相同key。
2. GameScene敌人配置查询从`ENEMIES`切换到`ENEMY_CATALOG`，但暂时不要修改当前波次。
3. 增加四种行为处理器；护盾需要独立盾值，爆破需要引信状态，毒雾和军医需要可清理的定时状态。
4. 增加目标优先级：爆破尸和军医高于普通目标，毒囊尸应优先在远端处理。
5. 行为验证后再将扩展类型写入`waves.js`，每次只替换等压力的普通名额。

## 风险

- 当前敌人对象没有护甲、护盾、状态效果与治疗字段，不能只把新类型写进波次。
- 毒雾和军医的定时器必须跟随暂停，并在死亡、清场和重开时销毁。
- 爆破尸门伤高，首次出现时每波最多一个；否则局部压力会超过当前第一关。
- 军医治疗应有单次上限，且不能治疗自己或Boss，避免形成无法击杀的循环。
